# 修改大逃杀配置

> 详细大逃杀配置Wiki：[配置文件说明](https://github.com/XColorful/BattleRoyale/wiki/Configuration-introduction)

## 配置设计

[自定义大逃杀](https://github.com/XColorful/BattleRoyale)并没有游戏内修改配置的设计，而是任意单个配置均支持提前准备任意数量的预设，并且所有配置项均可热加载，从而实现游戏内快捷切换，因此：
- 你应该在游戏外编写配置文件
- 如果默认配置（用[生成配置](https://github.com/XColorful/BattleRoyale/wiki/Example-command)生成）不够满足需要，你需要提前准备好足够数量的预设
- 如果实在需要临时微调，直接到配置目录修改对应文件，再[重载配置](https://github.com/XColorful/BattleRoyale/wiki/Reload-command)即可

## 主要配置项
> 对于任意[配置](https://github.com/XColorful/BattleRoyale/wiki/Configuration-introduction)，添加 _"default": true_ 会使其成为每次[重载配置](https://github.com/XColorful/BattleRoyale/wiki/Reload-command)时默认选中的预设

### 物资刷新
- [物资刷新配置](https://github.com/XColorful/BattleRoyale/wiki/Configuration-introduction#物资刷新配置)调整物资刷新器以及原版箱子的刷新内容
- [性能配置](https://github.com/XColorful/BattleRoyale/wiki/Performance-config)调整是否刷新原版箱子，以及刷新速度
> 如果遇到游戏玩家周围的箱子未刷新物资，通常是选用的[性能配置](https://github.com/XColorful/BattleRoyale/wiki/Performance-config)预设未开启原版箱子刷新或刷新速度过慢

### 大逃杀游戏规则
- [游戏规则配置](https://github.com/XColorful/BattleRoyale/wiki/Gamerule-config)调整队伍规模、大厅位置、自动饱和度、冒险模式等配置
- [出生配置](https://github.com/XColorful/BattleRoyale/wiki/Spawn-config)调整游戏开始时玩家的出生方式
- [区域配置](https://github.com/XColorful/BattleRoyale/wiki/Zone-config)提供了高度自定义的区域形状及功能、模组已经提供了经典大逃杀圈型预设
> [自定义大逃杀](https://github.com/XColorful/BattleRoyale)的核心就是高度自定义的[区域配置](https://github.com/XColorful/BattleRoyale/wiki/Zone-config)和[物资刷新配置](https://github.com/XColorful/BattleRoyale/wiki/General-loot-config)

### 客户端配置
- [显示配置](https://github.com/XColorful/BattleRoyale/wiki/Display-config)调整屏幕上队伍信息、生存人数等信息的位置以及隐藏
- [渲染配置](https://github.com/XColorful/BattleRoyale/wiki/Render-config)可使圆形/球体区域更光滑，并可以使新接收到的区域显示为本地定义的单一颜色

# English

> Detailed BattleRoyale Configuration Wiki: [Configuration Introduction](https://github.com/XColorful/BattleRoyale/wiki/Configuration-introduction#English)

## Configuration design

[Custom BattleRoyale](https://github.com/XColorful/BattleRoyale) does not feature in-game configuration modification. Instead, any single configuration supports preparing an arbitrary number of presets in advance, and all configuration items can be hot-reloaded, enabling quick switching in-game. Therefore:
- You should write configuration files outside the game.
- If the default configurations (generated by [Example command](https://github.com/XColorful/BattleRoyale/wiki/Example-command#English)) are not sufficient, you need to prepare enough presets in advance.
- If temporary fine-tuning is truly necessary, directly modify the corresponding file in the configuration directory, then [reload the configuration](https://github.com/XColorful/BattleRoyale/wiki/Reload-command#English)

## Main Configuration
> For any [configuration](https://github.com/XColorful/BattleRoyale/wiki/Configuration-introduction#English), adding _"default": true_ will make it the default selected preset every time [reload the configuration](https://github.com/XColorful/BattleRoyale/wiki/Reload-command#English).

### Loot Generation
- [Loot Generation Configuration](https://github.com/XColorful/BattleRoyale/wiki/Configuration-introduction#Loot) adjusts the content generated by loot spawner and vanilla chests.
- [Performance Configuration](https://github.com/XColorful/BattleRoyale/wiki/Performance-config#English) adjusts whether vanilla chests are generated and the generation speed.
> If chests near game players don't generate loot, it is usually because the selected [Performance Configuration](https://github.com/XColorful/BattleRoyale/wiki/Performance-config#English) preset does not have vanilla chest generation enabled or the generation speed is too slow.

### BattleRoyale Gamerules
- [Gamerule Configuration](https://github.com/XColorful/BattleRoyale/wiki/Gamerule-config#English) adjusts settings such as team size, lobby location, auto-saturation, and adventure mode.
- [Spawn Configuration](https://github.com/XColorful/BattleRoyale/wiki/Spawn-config#English) adjusts how players spawn at the start of the game.
- [Zone Configuration](https://github.com/XColorful/BattleRoyale/wiki/Zone-config#English) provides highly customizable zone shapes and functions. The mod already offers classic battle royale circle presets.
> The core of [Custom BattleRoyale](https://github.com/XColorful/BattleRoyale) is its highly customizable [Zone Configuration](https://github.com/XColorful/BattleRoyale/wiki/Zone-config#English) and [Loot Generation Configuration](https://github.com/XColorful/BattleRoyale/wiki/Configuration-introduction#Loot).

### Client Configuration
- [Display Configuration](https://github.com/XColorful/BattleRoyale/wiki/Display-config#English) adjusts the position and hiding of team information, alive player count, and other information on the screen.
- [Render Configuration](https://github.com/XColorful/BattleRoyale/wiki/Render-config#English) can make circular/spherical zones smoother and allows newly received zones to be displayed in a single locally defined color.
